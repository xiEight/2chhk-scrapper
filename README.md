# 2chhk-scrapper
<b>EN</b><br>
<h1>Simple script for scrapping multimedia from 2ch thread.</h1>
<b>How to use:</b><br>
Run script with youre IDE or on youre terminal:<br>
   <code>python3 scrap.py</code>
    
After lauching you need to put link to thread. Then the multimedia data will be downloaded from the thread to your computer. By default, the program downloads files to the current directory.
In order to change this, you need to run a script with an argument which contains the path to the necessary directory:<br>
    <code>python3 scrap.py /some/path/</code><br>
<b>Used libraries:</b><br>
<code>BeautifulSoup4
request
urllib3
</code>

<b>RU</b><br>
<h1>Небольшой скрипт для скачивания мультимедиа из тредов 2ch.</h1>
<b>Как использовать:</b><br>
Запустить скрипт можно через используемую Вами IDE или через консоль:<br>
   <code>python3 scrap.py</code><br>
После запуска введите ссылку на необходимый тред. После этого мультимедиа-файлы будут автоматически сохранены в текущую директорию. Для того, чтобы указать произвольный путь сохранения, передайте его в качестве параметра программы:<br>
   <code>python3 scrap.py /some/path/</code></br>
<b>Используемые библиотеки:</b><br>
<code>BeautifulSoup4
request
urllib3
</code>
